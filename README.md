# 100 Mexicanos Dijeron - Aplicación Android

## Descripción del Proyecto
Este repositorio contiene el código fuente, recursos y lógica de desarrollo de la aplicación móvil "100 Mexicanos Dijeron", desarrollada nativamente para el sistema operativo Android utilizando Kotlin.

El proyecto implementa arquitecturas complejas de software, incluyendo:
- Conexión Multijugador en Red Local (LAN) mediante Sockets TCP/IP y Network Service Discovery (NSD).
- Persistencia de datos local utilizando Room Database.
- Simulación de Inteligencia Artificial para el modo de un solo jugador.
- Integración de hardware (reconocimiento de voz y sensores hápticos).
